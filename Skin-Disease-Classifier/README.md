      <li>Leveraging Harvard's Ham10000 dataset and the robust MobileNet V2 model, this system boasts an impressive accuracy rate of 89%.</li>
      <li>But that's not all â€“ we've integrated "Derma" a dermatology chatbot powered by GPT-3.5 Turbo with internet access, to provide insightful information and guidance.</li>
