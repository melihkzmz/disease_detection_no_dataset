# WSL Hızlı Başlangıç

Script Windows dosya sisteminde (`/mnt/c/`). WSL'de doğru dizine git:

## Hızlı Başlatma:

```bash
# Proje dizinine git
cd /mnt/c/Users/melih/dev/disease_detection/Skin-Disease-Classifier

# Script'e izin ver
chmod +x start_training_bone_wsl.sh

# Başlat
./start_training_bone_wsl.sh
```

## Veya Tek Komutla:

```bash
cd /mnt/c/Users/melih/dev/disease_detection/Skin-Disease-Classifier && chmod +x start_training_bone_wsl.sh && ./start_training_bone_wsl.sh
```
